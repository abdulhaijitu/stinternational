 # ST International
 
 Scientific & Industrial Equipment Supplier - Bangladesh
 
 ## About
 
 ST International is a trusted supplier of scientific, laboratory, and industrial equipment in Bangladesh. Serving universities, research labs, and industries with certified products since 2005.
 
 **Website**: https://stinternationalbd.com
 
 ## Technology Stack
 
 - React 18 with TypeScript
 - Vite build system
 - Tailwind CSS with shadcn/ui
 - Cloud backend with authentication
 
 ## Development
 
 ```sh
 # Install dependencies
 npm install
 
 # Start development server
 npm run dev
 
 # Build for production
 npm run build
 ```
